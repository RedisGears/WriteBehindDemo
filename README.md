# RedisGears WriteBehind Demo

## Requirements

* Docker Compose
* About 64 CPU cores (mainly for simulating virtual users)
* Several GBs of free RAM
* A few GBs of free disk space

## Running it

```sh
git clone https://github.com/RedisGears/WriteBehindDemo
cd WriteBehindDemo
bash run.bash
```

## Stopping it

```sh
docker-compose down
```
